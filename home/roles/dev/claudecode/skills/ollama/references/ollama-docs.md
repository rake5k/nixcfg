# Ollama Documentation Reference

## Official Documentation

The primary source for all Ollama information is the official documentation:

**Main URL**: https://docs.ollama.com

## Documentation Sections

### Core Topics Covered in Official Docs

- **Getting Started**: Installation, setup, first steps
- **Models**: Model list, model details, model management
- **API**: REST API reference, authentication, streaming
- **Modelfiles**: Creating custom models, Modelfile syntax
- **Troubleshooting**: Common issues, error messages, solutions
- **Performance**: Optimizing performance, memory management
- **Deployment**: Running Ollama in various environments

## Verification Process

When answering Ollama-related questions:

1. **Consult the official docs** at https://docs.ollama.com
2. **Search the docs** for the specific topic
3. **Verify commands and examples** against the docs
4. **Check model availability** on the model list at https://ollama.com/library

## Model List

## API Reference

The Ollama API documentation is at:

**URL**: https://docs.ollama.com/api

The API allows you to:

- Run models
- Chat with models
- Pull/push models
- List models
- Create models from Modelfiles
- And more
